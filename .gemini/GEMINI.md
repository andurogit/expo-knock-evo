# project

- Policy Finder

## 구조

- React Native 사용
- DB 는 Supabase 사용
- Auth 는 Supabse 의 기본인 Mail 인증 방식으로 구성
- 하단 탭 이동 구조
- /app/(tabs) 구조로 개발

## 레이아웃 

- 기본적으로 회원 가입을해야 화면 진입 가능하도록 구성 
- 앱 초기 진입 시 노크하는 애니메이션 출력
- 이후 로그인 화면 노출 
- id/pw 입력창 
- 로그인/회원가입 
- 로그인 후 Home 화면으로 이동.

## 페이지 구성

- 하단 탭으로 구성
- Home, Search, My Page

### Home

- 최상단에 돋보기 모양의 입력 창으로 정책 검색 가능
- 정책 정보는 카드 형태로 아래로 스크롤 할 수 있는 형태
- 카드는 1row 만 보이도록 구성
- 카드에는 정책명, 위치, 설명 정보 (100자)
- 엠블럼?혹은 기관정보 표시

### Serch

- 현재 거주지 기준으로 정책이 있는지 검색
- 반경 조정 가능 1, 5, 10 단위 km
- 데이터 베이스 기준으로 존재하는 정책 지도에 표기


### My Page

- 회원가입 및 내 정보 입력 페이지
- 거주지 입력
